# 🎓 AI-Based Cheating Detection System

An AI-powered online exam proctoring system built using Computer Vision and Artificial Intelligence to monitor candidates during online examinations in real time.

The system detects suspicious activities such as:

* Multiple person detection
* Candidate absence
* Looking away from screen
* Suspicious head movement

The project uses webcam-based monitoring with OpenCV and MediaPipe for real-time facial and pose analysis.
